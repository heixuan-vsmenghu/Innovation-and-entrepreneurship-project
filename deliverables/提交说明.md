# 创策云 UniPlan AI 最终交付说明

## 一、需要打印提交的文件

- `deliverables/创策云UniPlanAI商业计划书.docx`
- 打印前需要补充封面信息：
  - 团队名称
  - 团队成员学号、姓名
  - 学院专业
  - 指导教师
  - 提交日期
- 打印前建议检查目录、表格分页、封面排版和附录说明。

## 二、课堂汇报要打开的文件

- PPT 文件：`deliverables/创策云UniPlanAI课堂汇报.pptx`
- 汇报讲稿：`deliverables/创策云UniPlanAI汇报讲稿.md`
- 前端原型目录：`prototype/`
- 如需现场演示原型，建议提前启动本地开发服务并打开浏览器页面。

## 三、问卷需要发放的链接位置

- 问卷星/腾讯问卷复制文本：`research/questionnaire_wjx_ready.md`
- 当前还没有正式问卷链接，需由市场调研负责人将题目复制到问卷星或腾讯问卷后生成链接。
- 问卷链接位置：【待问卷创建后补充】
- 问卷结果整理位置：`research/survey_analysis_template.md`

## 四、还需要人工补充的信息

- 团队成员学号、姓名
- 团队名称
- 指导教师
- 学院专业
- 提交日期
- 问卷真实结果
- 问卷链接
- PPT 中可替换为真实原型截图的页面图片
- 商业计划书中所有【待问卷调研后补充】位置

## 五、原型运行方式

```bash
cd prototype
npm install
npm run dev
```

启动后根据终端提示打开本地地址，通常为：

```text
http://127.0.0.1:5173
```

## 六、重新导出 Word 和 PPT

在项目根目录运行：

```bash
python scripts/export_business_plan_docx.py
python scripts/export_pptx.py
```

导出结果会覆盖更新：

- `deliverables/创策云UniPlanAI商业计划书.docx`
- `deliverables/创策云UniPlanAI课堂汇报.pptx`
- `deliverables/创策云UniPlanAI汇报讲稿.md`
